# 1. Contexto / Problema
A TLC tem 200 mil veículos gerando 1 milhão de viagens por dia, mas os dados históricos só são analisados trimestralmente em relatórios estáticos. Decisões de frota são tomadas com base em feeling, não em previsão. Isso custa dinheiro.

# 2. Objetivo
O que o projeto pretende alcançar.

Exemplo:

Reduzir o tempo médio de atendimento em 30%.

# 3. Diagnóstico / Evidências

Uma plataforma que atualiza automaticamente painéis de demanda por zona a cada hora, prevê a quantidade de viagens das próximas 24h com erro menor que 15%, e sinaliza viagens suspeitas em tempo real.
