# Pulse — Plataforma Analítica de Mobilidade Urbana em Lakehouse

Pulse é uma plataforma analítica end-to-end construída sobre arquitetura Lakehouse que ingere, processa, modela e expõe dados de mobilidade urbana de Nova York para gerar insights de negócio, dashboards executivos e modelos preditivos. O projeto simula o trabalho real de um time de dados moderno, cobrindo Engenharia de Dados, Análise de Negócios, Análise de Dados, Business Intelligence e Ciência de Dados sobre arquitetura Lakehouse moderna na AWS + Databricks.
